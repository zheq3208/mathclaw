## Isomorphic

Solve x^2 - 7x + 12 = 0

- Intent: Practice factoring monic quadratics with integer roots using the same method as the original problem
- Difficulty relation: about_the_same
- Coach note: This problem follows the exact same factoring pattern as the original. Find two numbers that multiply to the constant term and add to the linear coefficient.
- Changes:
  - Changed constant term from 6 to 12 and linear coefficient from -5 to -7 while maintaining integer roots (3 and 4)
- Answer outline:
  - Factor as (x-3)(x-4)=0
  - Set each factor equal to zero
  - Solutions: x=3, x=4

## Harder

Solve 2x^2 - 5x + 2 = 0

- Intent: Practice factoring quadratics with leading coefficients greater than 1
- Difficulty relation: about_the_same
- Coach note: When the leading coefficient isn't 1, use the 'ac method' or trial-and-error to find factors. Verify by multiplying your factors.
- Changes:
  - Introduced non-monic quadratic (leading coefficient 2) requiring additional factoring steps
- Answer outline:
  - Factor as (2x-1)(x-2)=0
  - Set each factor equal to zero
  - Solutions: x=1/2, x=2
