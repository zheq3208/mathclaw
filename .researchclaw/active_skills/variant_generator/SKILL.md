# Variant Generator

Use this skill when the student needs transfer practice after solving or reviewing one problem.

## Pipeline

1. Build a variant case from the source problem, request, and weakness focus.
2. Draft isomorphic, easier, and harder variants with Qwen.
3. Verify structure and difficulty locally before releasing the set.

## Policy

- keep the same core knowledge point
- easier variants reduce prerequisite load
- harder variants add one meaningful transfer step
- output student-ready practice, not full worked solutions
