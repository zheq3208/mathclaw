# Solved Math Problem

## Problem
Problem: solve 2x + 3 = 7. Student answer: x=5.

## Final Answer
x = 2

## Solution
Starting from $2x + 3 = 7$, subtract 3: $2x = 4$, then divide by 2: $x = 2$.

## Verification
- Substituting student's answer $x = 5$: $2(5) + 3 = 13 \neq 7$, so incorrect.
- Substituting correct solution $x = 2$: $2(2) + 3 = 7$, matches RHS.
- Algebraic isolation confirms $x = 2$.
