# Difficulty Calibrator

Use this skill to sanity-check whether a generated or imported problem matches the intended level shift.

## Consider

- concept count and prerequisite load
- algebraic or geometric reasoning burden
- diagram interpretation load
- whether the solve frame stayed aligned with the source problem
